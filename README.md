# NUK-student-interaction-website

這是以PHP為基礎開發的NUK學生宿舍活動網站，除了提供宿舍簡介、學生討論區、開Party以外，還有參考Dcard抽卡所延伸的家庭功能，可供使用者邀請夥伴組成家庭。

## Architecture

主要分為三個部分，「高大宿舍簡介、宿舍大小事、會員系統」。

會員登入系統分為一般使用者與管理員，管理員可進行照片編輯、刪除留言等功能，並可查看後台數據。
